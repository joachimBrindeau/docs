# FAQ for Voxel Addon

## Frequently Asked Questions

### 1. What is the Voxel Addon?
The Voxel Addon is a WordPress plugin designed to enhance your website's functionality by providing advanced features and tools for managing voxel-based content.

### 2. How do I install the Voxel Addon?
To install the Voxel Addon, follow these steps:
1. Download the plugin from the WordPress repository or the official website.
2. Go to your WordPress admin panel.
3. Navigate to Plugins > Add New.
4. Upload the downloaded plugin file and activate it.

### 3. What are the system requirements for the Voxel Addon?
The Voxel Addon requires:
- WordPress version 5.0 or higher
- PHP version 7.2 or higher
- MySQL version 5.6 or higher

### 4. How can I get support for the Voxel Addon?
For support, you can visit our [support page](#) or contact our support team via email at support@without-code.com.

### 5. Are there any known issues with the Voxel Addon?
Currently, there are no major known issues. However, if you encounter any problems, please refer to the [troubleshooting guide](/partials/troubleshooting) or contact support.

### 6. Can I contribute to the Voxel Addon?
