# Getting Started with Voxel Addon

Welcome to the Voxel Addon documentation! This guide will help you get started with the Voxel Addon plugin for WordPress.

## Prerequisites

Before you begin, ensure you have the following:

- A WordPress installation (version X.X or higher)
- Access to the WordPress admin dashboard
- Basic understanding of WordPress plugins

## Installation

1. **Download the Voxel Addon Plugin:**
   - Obtain the plugin files from the official repository or your local development environment.

2. **Upload the Plugin:**
   - Go to your WordPress admin dashboard.
   - Navigate to **Plugins > Add New**.
   - Click on **Upload Plugin** and select the downloaded Voxel Addon ZIP file.
   - Click **Install Now**.

3. **Activate the Plugin:**
   - After installation, click on **Activate** to enable the Voxel Addon plugin.

## Configuration

Once activated, you may need to configure the plugin settings to suit your needs. Navigate to the Voxel Addon settings page in the WordPress admin menu to customize your options.
